Quick Start

How it works

Three steps from messy requirements to structured spec.

Drop your requirements

echo "Users should log in..." > source/login.md

Any format: Markdown, TXT, chat logs. Put them in source/.

Run the agents

spec-editor run

Agents debate and produce a structured specification in aspects/.

Connect to your IDE

spec-editor mcp &

Claude Code, Cursor, or Zed now reads your spec for context.
